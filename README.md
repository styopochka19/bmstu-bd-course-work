# Курсовая по БД

## Описание проекта

Это десктоп-приложение, созданное для курсовой работы, на Python для записи пациентов на прием к врачам в медицинских учреждениях. Приложение предоставляет функционал регистрации пациентов, записи на прием, просмотра медицинской истории и оценки качества услуг.

## Функциональные возможности

### Аутентификация и регистрация
- Регистрация новых пациентов с валидацией данных
- Аутентификация зарегистрированных пользователей
- Хранение данных пациентов в базе данных

### Запись на прием
- Выбор специализации врача
- Календарь для выбора даты приема
- Просмотр доступных врачей с информацией о рейтинге и больнице
- Выбор времени приема из доступных слотов

### Личный кабинет
- Просмотр персональной информации
- Доступ к медицинской карте
- История записей на прием
- Просмотр истории госпитализаций

### Система оценок
- Оценка врачей после приема
- Оценка медицинских учреждений
- Расчет средних рейтингов

## Технические особенности

### Используемые технологии
- **GUI**: CustomTkinter для интерфейса
- **База данных**: Microsoft SQL Server через pyodbc
- **Календарь**: tkcalendar с кастомными модификациями
- **Валидация**: регулярные выражения для проверки данных
